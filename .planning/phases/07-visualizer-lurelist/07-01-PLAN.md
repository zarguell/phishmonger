# 07-01: Update Visualizer Card Layout

**REQ:** ANN-12  
**Phase:** 7  
**Status:** Pending

## What This Is

Update the visualizer card (annotation display) to show: title → tags inline → description.

## Implementation

1. Locate visualizer card component (likely `AnnotationCard` or similar)
2. Restructure card content:
   - Line 1: Title (bold, if present)
   - Line 2: MITRE tag and Persuasion tag (if present, inline)
   - Line 3+: Full description text
3. Handle edge cases:
   - No title, no tags: just description
   - Title + tags: title first line, tags second line
   - Tags only: no title, tags on first line, description below
4. Test with various annotation combinations

## Files Likely Affected

- `src/components/AnnotationCard.tsx` — Restructure display
- `src/components/Visualizer.tsx` — Card rendering
- `src/styles/visualizer.css` — Card styling
