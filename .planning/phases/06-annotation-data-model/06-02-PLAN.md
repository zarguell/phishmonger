# 06-02: Add Optional MITRE ATT&CK Tag to Annotation

**REQ:** ANN-10  
**Phase:** 6  
**Status:** Pending

## What This Is

Allow annotations to optionally reference a MITRE ATT&CK technique. The tag displays as "T1598 - Phishing for Information" style.

## Implementation

1. Add optional `mitreTag?: string` field to `Annotation` type
2. Create tag display component that formats: `T{ID} - {Name}`
3. Add MITRE technique selector to AnnotationPanel (from existing library)
4. Ensure tag can be cleared (optional, not required)

## Files Likely Affected

- `src/types.ts` — Add `mitreTag?: string` to Annotation
- `src/components/AnnotationPanel.tsx` — Add MITRE selector
- `src/components/Tag.tsx` (create if not exists) — Tag display component
- `src/data/techniques.ts` — Existing MITRE library

## Tag Display Format

```
(T1598 - Phishing for Information)
```

Should be styled as a small inline tag, not bold.

## Success Criteria

- [ ] Annotation optionally stores MITRE technique ID
- [ ] AnnotationPanel allows selecting MITRE technique
- [ ] Selected technique displays as formatted tag
- [ ] Tag can be cleared/removed
- [ ] Visualizer shows MITRE tag inline (ANN-12)
