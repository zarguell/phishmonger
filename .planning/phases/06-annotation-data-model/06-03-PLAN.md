# 06-03: Add Optional Persuasion Principle Tag to Annotation

**REQ:** ANN-11  
**Phase:** 6  
**Status:** Pending

## What This Is

Allow annotations to optionally reference a Cialdini persuasion principle. The tag displays alongside MITRE tag.

## Implementation

1. Add optional `persuasionTag?: string` field to `Annotation` type
2. Add persuasion principle selector to AnnotationPanel (from existing library)
3. Create formatted display: `(Persuasion: {Principle})`
4. Ensure both MITRE and Persuasion tags can coexist

## Files Likely Affected

- `src/types.ts` — Add `persuasionTag?: string` to Annotation
- `src/components/AnnotationPanel.tsx` — Add persuasion selector
- `src/data/persuasion.ts` — Existing persuasion principles library
- Tag display component from 06-02

## Tag Display Format

```
(Persuasion: Urgency)
```

Both tags appear on same line in visualizer card.

## Success Criteria

- [ ] Annotation optionally stores persuasion principle
- [ ] AnnotationPanel allows selecting persuasion principle
- [ ] Persuasion tag displays formatted inline
- [ ] Both MITRE and Persuasion tags can be present
- [ ] Tags can be independently cleared
