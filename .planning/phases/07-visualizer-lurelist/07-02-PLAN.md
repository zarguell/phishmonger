# 07-02: Update Lure List to Show Title/Description

**REQ:** ANN-13  
**Phase:** 7  
**Status:** Pending

## What This Is

Update the lure list sidebar to show annotation title (or description preview) instead of UUID.

## Implementation

1. Locate lure list component (`LureList.tsx`)
2. Update list item display:
   - If annotation has title: show title
   - If no title: show first ~30 words of description
3. Truncate long titles/descriptions with ellipsis
4. Keep lure number badge on left

## Files Likely Affected

- `src/components/LureList.tsx` — Update item display
- `src/components/LureListItem.tsx` — Individual item component

## Lure List Item Display

```
[1]  Title (if exists)
     First few words of description...
```

Or if no title:
```
[1]  First few words of description...
```

## Success Criteria

- [ ] Lure list shows title instead of UUID
- [ ] No-title annotations show description preview
- [ ] Long text truncated with ellipsis
- [ ] Clicking item still scrolls to lure in preview
- [ ] Visual consistency with rest of UI
